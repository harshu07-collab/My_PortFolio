# Animated 3D Portfolio

This is a high-performance, animated portfolio website built with React, Three.js, GSAP, and TypeScript, inspired by the tutorial by Rajesh Chityal.

## 🚀 Features
- **3D Avatar**: Interactive Ready Player Me avatar that rotates as you scroll.
- **Smooth Animations**: Powered by GSAP and ScrollTrigger.
- **Modern UI**: Clean and minimal design.
- **Fully Customizable**: Easy-to-edit configuration.

## 🛠️ Customization
To customize the portfolio with your own information, edit the file:
`src/constants/index.ts`

You can change:
- Your Name
- Role
- About Me text
- Email
- Social Links
- Projects (Title, Description, etc.)

## 📦 Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## 🎨 3D Avatar
The avatar uses a Ready Player Me model. You can change the model URL in `src/components/Avatar.tsx`.
To create your own avatar, visit [Ready Player Me](https://readyplayer.me/).
